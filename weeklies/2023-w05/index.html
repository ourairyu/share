周报 2023W05：「开工」这一周我都做了啥

欧雷 发表于 2023-02-04 14:28:53

年后开工一周了，这也意味着我有三周没工作、没收入了，看着隔三差五的固定支出账单，心里难免有些慌慌的，焦虑油然而生。

艰难？挺着！

每年年末到次年年初这期间，很多按年扣款的账单，并且不少金额还不算小，像保险、域名（有好几个）、云服务器、阿里小号等等。不知道工资卡里那几万块钱能够坚持到什么时候……😮‍💨😮‍💨😮‍💨

然而，越是这种艰难时刻，越要调整好自己，保持沉着、冷静的心态，这样才能避免掉进过往的陷阱，才有希望真正地破局。我曾说过「自由与风险并存」，当时只是逻辑推断，现在则是亲身经历，感受更为深刻。

一个动物，如果是被圈养的，吃住这类与生存息息相关的事情它可以完全不愁，吃啥喝啥与住在什么环境只能接受，并且不一定哪天就被宰了；而野生在外的，虽说可以到处跑并找自己想吃的喝的，但也得找到才行，吃一顿饿好几天都是很正常的，生死掌握在自己的能力和自然规律的手中。

我现在就是处于野生状态，至于以后会不会再乖乖地回去被圈养我不知道；反正，先野一段时间再说！

同步「开工」

大家都在上班努力赚钱，我自然也不敢怠慢，甚至还要更用功——我得尽快找到能够让自己和🐷能正常生存的手段呀！不同的是，我是完全在做自己的事，若是找到自己的模式了，效果会成倍地放大。

没有特殊情况的话，每天我都是上午七点半到八点半之间、下午一点半到两点半之间开始做事，晚上五点到七点之间吃饭，吃完可能还会继续做到八九点——跟在上班时的作息差不多。

这周精力主要放在完善并更新我的网站上了，包括相当于是「我」这个「产品」的「产品介绍手册」的个人主页 https://yaol.in/ 和发表我的作品和日常生活状态的个人网站 https://ourai.ws/——

先在个人主页上更新了当前的工作状态，并添加/更新了「履历」和「合作」页面：「履历」中罗列了我对以往工作经历和不同类型个人产品的简介，每段工作经历都会跳转到相应的复盘文章页面；「合作」中对我的个人能力、可以合作的方向做了简单介绍。

很是凑巧，就在我把个人主页的功能和内容弄得差不多后的第二天，在群中看到了一个符合我合作方向的兼职（临时工）工作，就将还热乎的「履历」与「合作」页面网址甩了过去。

之后在个人网站上添加了「开源」和「赞助」页面：「开源」中罗列了我自己发起和参与贡献的开源软件项目；「赞助」中则介绍了如何对我进行经济上的支持以使我能够持续创作并为更多人甚至是行业、社会提供价值。

另外，还在每篇文章的末尾加上了「充电」提示：

可以说，这周所做的事情是为我的个人品牌形成最小闭环做准备。

重启「暴走杭州」

多年前未完成的「暴走杭州」要重新捡起来了：

周二那天，难得的天气晴朗、阳光明媚，并且气温符合🐷的要求，于是乎我们在午饭过后就收拾好基础物品与装备，开车到汽车西站，打算从法华寺那里上山去北高峰。

这条路线多年前我走过，因其陡峭漫长，十分考验毅力、耐力和体力——

目前我所走过的杭州的山路当中，最折磨人的是屏峰山上山，其次是老东岳法华寺附近上山，第三是灵山风水洞附近上山。对自己的毅力、体力有信心的可以去挑战下。

——欧雷《劳动节小记之白龙出渊》

在上山途中，我喝哧带喘地走走停停了半个钟左右才到上面，而🐷明显看起来比我轻松，明明显得那么瘦弱的一只🐷……一定是新冠的影响！（无脑甩锅，我才不承认自己老了呢！哼！）

上去之后再走没多久就到财神庙了，看到那人山人海地参拜，撞钟，拍照，给我一种错觉——这是还没开工呢吗？这些人都是老板吗？

我们在原路返回还是从灵隐下山之间经过片刻纠结，最后选择了后者。索道那里排满了人，我们徒步下山（本来也没考虑坐缆车）。

下山路比上山时平缓多了，轻松了很多，我一点都不累；看着其他从灵隐上山的人龇牙咧嘴地叫苦，真想让他们体验一下我上山时的痛苦。

下到山脚后的时间正好是下班高峰期，公交车上会挤满了人，并且🐷担心我又像之前那次没什么补给走到上天竺那样难受呕吐，想跟我一起经灵溪隧道走回到汽车西站。

最终我们总共花了两个半小时左右，运动带来的后劲儿使我们腿（肌肉、筋）痛了三天多才缓过来。为了身体健康和「暴走杭州」的顺利进行，我们初步决定先尽量每两周出去与大自然亲密接触一次。

接下来计划

暂时也没什么中远期计划，因为有很多事情对我来说是新鲜的、未知的、充满不确定性的，只对这个月大概做啥有点想法——

下周主要是先把 Handie 的 React 版本的核心升级到最新，修复下优先级比较高的 bug 并同步给前同事；下周剩余时间及之后几周的时间，去调研下自由职业者的商业模式，同时开始在知乎上寻找合适的问题用心作答。

只有在了解了「都能做啥」以及「我能做啥」，再与「我想做的」相匹配融合后才能制定出中远期计划。

前几天在浏览 Vue 官方文档时无意间看到个提示：

Vue 2 将于 2023 年 12 月 31 日停止维护。详见 Vue 2 延长 LTS。

看过之后我心想：「这还真是 Vue 官方给我找活儿干啊！」这样一来，Handie 的 Vue 版本也要尽快支持 Vue 3.x 了，因为当前是用 Vue Class Component 写的。

不过，虽然重要，但不紧急，下半年再说吧！没准儿那时我已经有了个很好的跨视图库/框架的 UI 组件源码编写方案，可以很轻松地撸一遍。

最后再来几句

最近几个月的心境，还比较喜欢看那些认真活着的人们为自己生活、工作上的事情所恼有感而发的文字，再加上去年的年总结中说那是我最后一次以那种「俗套」的方式写「年总结」，以及几天前看了漩涡的博客，我想要不干脆把我的周报也都再次公开算了！

对我来说，周报和写文章不一样——周报是以时间为维度叙述，内容是一周内的事情，不一定在同一个方面；而文章是以观点、主题为维度，内容都是围绕着同一个观点、主题而展开的——因而文章是创作，周报不是。

以我现在的习性而言，周报可能会更能看到我的生活气息，有人情味儿些；而文章大多都是在严肃且认真地进行反思并探讨某个问题，像个冰冷的批判机器。

但是，我也不知道这个公开周报的想法能够坚持多久，正如俺家🐷前两天说我的：「你的想法总变！」

最后的最后，说句与上文所有内容都不相关的——《迷托邦》才玩不到一个月，游戏时间竟然已超买了十多年的《超级马里奥 3D 大陆》！
