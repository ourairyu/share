周报 2023W39：这摊儿不摆了

失业的第 37 周，又有 10 个星期没写周报了。

自从 8 月开始搞「破局之路」这个付费文章系列之后，每周都会写一篇文章进去，感觉那里正渐渐成为真正的「周报」……

完善「反混沌」网站

本来打算这周把《JavaScript模式》看完，但看到自己「反混沌」网站的首页空荡荡的，就手痒想把一直要弄的文章功能给加好。

之前有段时间我每天会在一个群里发篇符合「反混沌」思想的文章作为「每日阅读」，然后写进 Markdown 文件里作为记录提交到 Git 仓库中。

当时想弄网页，一方面是为了更方便查看记录，另一方面是可以充实网站内容。

我要在网页中显示的话，首先得写个脚本从 Markdown 文件中提取数据以结构化的形式保存到 YAML 文件里，并将转换后的结构化数据作为数据源。

这摊儿不摆了

我们从一两个月前就开始清库存了，也就是尽量不新进原材料，而是只去消耗；若实在还有剩余，能转卖就卖出去，不能就扔了。

虽然天气转冷占一点原因，但更多的是🐷终于认清了现实——卖这玩意又累又赚不了什么钱。

起初刚卖时我就不看好，除了人工投入没其他门槛，天花板极低，换我才不会选择去做；可她偏偏觉得这做的人很少，有搞头。

既然是她的「事业」，我也不想太强硬地去干预，就想着让现实去教她——很少人去做，是有原因的。

六七月份处于梅雨季，总下雨，就没怎么摆摊；最近两个月，🐷在网上疯狂地寻找免费和有补贴的集市，偶尔会有线上订单。

前几天，我们在外面刚送完货，她得知良渚那边国庆期间的集市报名没有通过，就变得阴郁起来，回到家后也不怎么说话。

那时她因为准备线上订单而已经连续两三天没怎么休息了，本来身体就差，国庆期间再劳累的话要吃不消进医院了。

我也是如此安慰她的，想让她放松心情多休息，别那么在意；就算从综合的得失层面考虑，也不值得啊！可是她听不进去……

忘了后来是什么契机，我跟她聊起了道家思想——她之前还总发一个讲庄子的视频号给我看，是时候「反击」她了！

我问她：「你知道道家的核心思想是什么吗？」

她答道：「无为？」

我说：「这不是最根本的，其实都已经在名字里了，就是『道』。」

我又说：「不说那些什么本质之类的东西，就简单理解，『道』就是规律，自然的那个规律，其他的『无为』什么的都是以此为基础的延伸。」

我再说：「道家讲究的就是遵循规律，万事万物都有自己的演化路径，不必去故意干涉，顺其自然。发生了的事情就不用去纠结了。」

我继续说：「人之所以会痛苦，是因为跟他人进行比较，从而产生一些欲望，困扰自己。假如世上只有你一人，你还会为那些事烦恼痛苦吗？所以，要向内求，放过自己，别去管外界怎样。」

再进一步：「就像人死了，没什么可悲伤的，只是又回归大道了，脱离了人间的痛苦，是件值得高兴的事。你看庄子他老婆死了，也只是刚开始悲伤那一下，后来又为其鼓乐。」

我说嗨了：「人出生时是精子与卵子结合形成，死后参与到自然循环成为微生物的养料，而灵魂会回归宇宙，来这世上只是走那么一遭，一切都顺其自然，没什么可在意的。」

没想到，平时基本不爱听我讲大道理的🐷，这回听得很是认真，没过多久她就从阴郁中走了出来。
