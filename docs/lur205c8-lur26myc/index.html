测听力

之前有次跟🐷去山姆时，发现可以免费测视力和听力，正好那时她的眼镜坏掉而想要配副新的，就去测了视力，并想让我测下听力。

在还年轻时（大概 14、15 年左右），我很喜欢戴着耳机开很大声听摇滚、重金属，且总用的是入耳式耳机；当时听了一段时间后感到自己听力下降了，就不怎么戴耳机听了。

再加上平时她或别人突然跟我说话时总是听不清（不确定）说了啥，总会让其重复一遍，这种种体验真的让我们都觉得我的听力不好了。

那次去山姆因为要等很久就没测听力，今天那里没人，我就测了。

测的方式很简单，就是左右耳都戴上入耳式耳机，会分别在一边播放分贝数很小的声音，听起来像鬼片背景音乐那种……只要听到声音就要按下手里的按钮。

由于音量实在是太低了，有时我觉得自己听到声音了，但又怀疑是不是幻听，就没去按……

最终测试出的结果，我的听力灵敏度是没问题的：

听不清别人说话可能是因为注意力的问题。

要做没成

昨晚发现卫生间水池下方地面有滩水，问🐷之后，她说自己没弄水，打开柜门一看，原来是水管漏了……

若是换做以前的话，遇到这种破事儿心里肯定开始烦躁了，但自从去年被厨房下水管漏水收拾得服服贴贴后，这种小事儿完全无法撼动我的内心！

经排查，是水管连接处那块老化裂开了，并且观其构造并没有厨房下水管那般复杂，就等着俺家🐷买的管子到货后换掉就好了。

可当今天我换完试水时才发现，上面跟水池子连接的地方也漏水，这下又得买上半部分等明天到货了才能彻底解决问题……

应该能……顺利解决吧？？？

另一件打算今天做但没成的事就是，本来约了今天跟两位前同事见面聊天，但由于杭州这雨下得有点大，改成明天了。

希望明天这两件事都顺顺利利～
