与前同事碰头

今天杭州天气很好，好得我都有点热了，尤其是下午出门的时间，应该是一天中最热的时候；我们约在黄龙那见面，由于我住在郊区中的郊区，到那乘公交转地铁要一个小时左右。

我在古荡出站，走几步后看到有架人行天桥凌空穿过天目山路，应该是最近这两年才建的，印象中多年前是没有的。

当看到许久未见的师兄与师弟时，心情是愉悦、激动的——距离上次与师弟见面刚好一年多点，而师兄则有两年甚至更久。

于我而言，这次相见的意义除了久别重逢之外，最重要的是想弄明白师兄当前在做的到底是怎样一件事，从而思考我能从什么角度切入去辅助——正如我在建设 Nop 官网与社区。

我们起初是在楼下边转边聊，当聊到 Coze 时，师弟说他们公司开源了个 AI 应用层项目，想演示给我们看下，就借他光踏进了传说中的 PingCAP。

上去之后我们又聊了好几个小时，最终可以得出核心结论——师兄做的基本是 BFF 层，我可以「毫无顾忌」地着眼于纯前端的表现层，即进一步建设 Petals、Handie。

另外，我们都认为 AI 必会取代 CRUD 工程师，既然这样，那就由我们去加速这个过程——

我要反思「Web 前端」是想让它「死」得更快一些，加速周期的迭代进程——促进「反混沌前端工程」的发展，迎接知识驱动的、智能的产研一体化平台。

欧雷《是时候反思下 Web 前端了》

我们到楼下麦当劳简单吃了个晚餐就各自道别了。

今天的聊天对我很有价值！
