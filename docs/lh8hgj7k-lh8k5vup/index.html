我们的创业

欧雷 发表于 2023-05-04 10:03:20

摆脱他人控制，实现自由生活。

基本共识

为了创业活动的顺利开展，一些基本前提需要两人达成共识。

目标

两人都处于没工作状态，纵有一定积蓄，可每月、每年有不少固定支出，如此以往终有弹尽粮绝之日。

对于基于雇佣关系的全职工作，林在生理与心理层面双重抗拒，觉得是精神凌迟；姜虽能接受，但领着死工资无任何增长空间，浪费时间与才能。

我们都想过不受他人控制的自由生活，故选择各自做（比较）感兴趣且擅长的事情去创业。

以下几个为两人各自的阶段性目标，即每人都按顺序经历一遍：

  1. 回本并盈利；
  2. 覆盖自己的社保；
  3. 覆盖自己的重疾保险；
  4. 覆盖家庭生活的平均月支出。

下一个目标是在上个目标的基础上进行累加，如「覆盖自己的重疾保险」的意思是「平均月利润至少是自己的社保与重疾保险所需金额的总和」。

当两人都达成上述几个目标时，家庭积蓄的损耗将会富有成效地被遏止，离数字游民的自由生活不远了！

业务

以虚拟产品/商品为主，实体商品为辅；在相对垂直的领域销售有所关联的商品，扩展出多种收入方式。

产品/商品

林主要是虚拟产品/商品，如知识及相关服务、软件及相关服务，收入结构大致如下：

  • 佛系付费（赞助与打赏）
    • 入门/普适性图文与视频创作

品牌

塑造品牌是为了促进销售并提高销量，给未来拓展品类和收入方式等铺路，最终目的是增加利润；在当下这个时代，塑造品牌所花成本最低的方式应该就是通过运营自媒体了。
