三五而已

欧雷 发表于 2023-02-21 17:47:28

算上这篇，我已经连续两年在自己周岁生日的时候写与年龄和人生有关的文章了，在外人看来这仿佛变成了惯例，但我不希望被如此认为，故特此说明一下——

之所以会在这样的时间点去写，是因为要写的内容与固定周岁的关联性很大，并且对我很有意义。

年龄焦虑

一提到「年龄焦虑」，貌似都是女人的事儿，跟男人没啥关系，毕竟社会对女人的评价几乎都在容貌、身材等外观条件上；并且从生性上来说，男人/雄性也确实本能地更容易被年轻貌美有活力的女人/雌性所吸引。

但并非只有女人才有年龄焦虑，男人也有，只不过不像女人一样是在外貌上，而是在事业、权势等方面——这同样是既受社会环境影响，又是刻在基因里的本能。

我在三十岁之前，虽然不是那么严重，但也在承受着年龄焦虑的困扰。

那时的我认为，自己在三十岁左右时的状态，应当是在一家前景不错的公司或团队中作为举足轻重的核心成员、中坚力量存在，有着满意的薪水、成长空间和影响力。

然而实际情况是，靠努力能实现的部分差不多达到了，占更大比例的剩余部分我无能为力，最终结果是不太符合我对自己三十岁的期望。

过了三十就是三十五，过了三十五又有四十，如此往复，直到退休并死亡，要一直为了几年后自己该是什么样而焦虑吗？到了几年后若没达到期望又去后悔前几年要是怎样也许就不会没达到？

焦虑又有什么用呢？焦虑就是无法认清自己的内心和所谓的「现实」而带来的落差所导致的——我多次说过，不必要的精神需求是让自己不幸福的根源。

在到了三十岁之后，我跟自己和解了，不再给自己制定那种令人难受、痛苦的荒唐目标，之后就再也没因年龄问题而焦虑。

年龄歧视

一个人自打呱呱坠地开始就受到年龄歧视——他还听不懂人话，像小动物似的，做错事就该打；小孩子乳臭未干，未经世面，懂什么；年轻人做事心浮气躁，不靠谱，还得是老人稳重；那个老古董跟不上时代的节奏，早该被淘汰了。

年龄歧视是无处不在的，就算是再平等对待他人的人，也会在某个不经意间对别人进行年龄歧视。

若只是日常生活中单纯的个人层面的看法，也不会有多大的影响，基本只会让被歧视的人心里不快后发生争吵；但要是成为社会层面的问题，就会侵犯到广大人群的利益，需要政府政策与法律法规的保障。

社会群体按年龄阶段可分为青少年、中年和老年这三类。

从国家角度来看，青少年和中年是国家发展所必要的，前者是精力充沛并充满活力的新生力军，后者则是精力与经验兼备的中流砥柱；而老年人说难听点就是累赘，毕竟这个阶段的人不太能为国家和社会创造什么价值了。

另外，站在企业的立场，尤其是那些提供虚拟服务的企业，更是不会把老年人纳入他们所服务的目标群体当中，各种 app 对老年人都不太友好，甚至是那些日常生活出行必备的。

老年人是社会的弱势群体这点是毋庸置疑且很难改变的事实，只能期望每个不是老年人的人能够由衷自发地对他们更加尊重——在平日里耐心对待他们的同时，作为企业员工在进行产品设计与研发时多考虑下他们该如何去使用。

但本当是社会中流砥柱的中年人呢？他们虽然在精力上很可能不如青少年那般旺盛，但掌握着绝大部分青少年所不具备的从多年实践经验中提炼出的实用的知识与智慧。

由于企业运作的特点，不需要太多经验丰富的人，反而因生产工具不够先进而需要大量拧螺丝的螺丝钉——精力充沛并充满活力的青年人正符合。

并且，在国内的环境下，中大型企业试图垄断，稍具创新能力的小企业很难获客并生存，对中年人来说相对稳定且薪资水平在可接受范围内的小企业工作机会较难获得。

再加上企业招人对（原始）学历要求越来越高，而当前是中年人的群体在（原始）学历上普遍达不到那么高的要求——年龄与（原始）学历带来的双重歧视，让中年人更难找到合适的工作机会；若求职者是中年女性，将会再次叠加性别歧视带来的暴击……

如果国家再不在政府政策与法律法规上给予中年人足够的保障，我认为社会将会朝着很糟糕的方向发展。在解决人口问题之前，难道不应该优先解决就业问题吗？

不出意外，每个人都会成为中年人，再变成老年人，不应在自己还相对年轻时就对他们不管不顾，反而更应该去努力为他们谋福利——量变而质变，帮助他们就是在帮助未来的自己！

对抗歧视

他人的歧视侵犯了我们本该享受的权益，作为受害者必然要为维护自身权益而奋起反击！然而，年龄歧视是社会结构性问题，单凭几个人是无法做出什么有效改变的，还是得靠国家从宏观层面出手。

鉴于此，与其去对抗或反击，不如认识并接受不确定性后正面且认真地去面对——拥抱变化。毕竟我们很难改变环境，而更容易改变自己以适应环境。

我不太担心自己变成老年人后会被歧视，因为我的思想还算开放，能够接受并理解新鲜事物，到时应该不会被时代的发展所落下；反倒是踩着「三十五岁红线」的当下，更该去思考如何才能有效应对危机的来临。

随着对国内环境了解的深入，我越发觉得打工是没有前途的——这并非是抱怨，也不是制造焦虑，而是我所认为的「现实」。也许有人会说：「你觉得打工没前途，那去创业啊！」

创业确实是条路，但正像上文所说，在国内走这条路必定会十分艰辛，很可能不仅没赚到钱，还把自己的底裤赔掉——方向、市场、政策、资金、机遇等缺一不可！

我更看好也更倾向于走自由职业这条门槛相对较低的路，顶多再找几个情投意合的人一起开个基于远程协作的工作室。

在自由职业这方面，有人爱捞偏门赚快钱，也有人喜欢贩卖焦虑后割韭菜，这都无可厚非，而我要秉持长期主义打造自己的产品，除了赚钱糊口也要为社会带来正向影响。

我自认为自己还是很适合做自由职业的：

  • 专业能力和业务爱好天然与自由职业亲和，转变成本较低；
  • 好奇心比较重，有持续学习的心态，易于发现新事物；
  • 思维比较开放，并对人性、社会和世界等方面有较为深入的思考，容易接受并理解新事物；
  • 观察力、想象力、执行力、自我管理能力等都还不错；
  • 懂得平等互利、合作共赢。

要想做自由职业，首先得找到合适的市场，需要观察并发掘需求，且自己在这方面有较大优势。那么，对我来说，做什么比较合适呢？

上文中说了，大多企业需要的是有活力的青年人充当拧螺丝的螺丝钉，而大量经验丰富的中年人无处可去——

这就导致中年人从多年实践经验中提炼出的实用的知识与智慧不能被很好地传承，随着中年人的失业而消失；当那些「青年人」也同样从自己的经验中提炼出来之时，他们很可能也已步入中年。

从这看似是死循环的局面中其实隐含着一个跳出条件——中年人可将自己本来用作生产工具的知识与智慧包装成产品，售卖给需要的青年人；不仅知识得以传承，青年人可让自己少走弯路快速成长，还能一定程度上解决中年人的收入问题。

上述方式可作为我的知识产品，除此之外，还打算做一些软件产品，主要是以下领域和方向：

  • 以 Web 相关技术为基础的软件供应链、研发提效；
  • 去中心化的以个人为中心的服务。

这两类之前已经说过无数次了，之所以多次述说，可表达我坚定的意志。

总结

本文分别粗浅地分析了人们普遍会面临的年龄焦虑与年龄歧视，并阐述了我（已经或准备）大概如何去应对它们。

社会给人们套上了太多的枷锁，没意识到的人们不自觉地顺从，而那些意识到了的人们，一部分不知怎样去挣脱而感到痛苦煎熬，另一部分则获得了自由而悠然自得。

认清现实不是为了妥协，而是更好地追梦。オレは負けない。

欧雷的想法

我才三十五而已，纵使被社会逼到墙角，只要我不放弃，必然会绝处逢生！
